En este proyecto desarrollamos el análisis de datos para una Tienda Online ficticia del rubro tecnológico. El objetivo fue generar, procesar y analizar datos de ventas para obtener información útil que ayude a la toma de decisiones de negocio.

Se utilizó:





Python para la generación de un dataset sintético y preparación de los datos.



MySQL para el modelado relacional, carga de datos y consultas para exploración y validación.



Power BI para el desarrollo de un dashboard interactivo para visualizar métricas claves de venta, comportamiento de clientes y tendencias temporales.

Este proyecto integró todo el flujo de trabajo de análisis de datos:

Generación —> Modelado —>Exploración —> Visualización —>Creación de estrategias de negocio
